
Feature List:

-   🌟 Tech stack: MERN + Socket.io + Chakra UI
-   🎃 Authentication & Authorization with JWT
-   📝 Create Post
-   🗑️ Delete Post
-   ❤️ Like/Unlike Post
-   💬 Comment to a Post
-   👥 Follow/Unfollow Users
-   ❄️ Freeze Your Account
-   🌓 Dark/Light Mode
-   📱 Completely Responsive

### Setup .env file

```js
PORT=...
MONGO_URI=...
JWT_SECRET=...
CLOUDINARY_CLOUD_NAME=...
CLOUDINARY_API_KEY=...
CLOUDINARY_API_SECRET=...
NODE_ENV=dev
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
